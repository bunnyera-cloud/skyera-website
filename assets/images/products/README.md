# SkyEra Product Image Placeholders

These SVG files are temporary product placeholders used by the homepage and product detail pages.

Replace each file with the final product image when ready. Keep the same filename if you do not want to edit HTML paths.

## Current image files

- `vision-pro.svg` - Apple Vision Pro
- `h50.svg` - Xiaomi Robot Vacuum H50
- `audio-glasses.svg` - MIJIA Smart Audio Glasses
- `ps5-controller.svg` - PS5 DualSense Controller
- `mini4pro.svg` - DJI Mini 4 Pro / Mini 4 Pro Drone

## Recommended final image format

- Use `900 x 700` or a similar 9:7 ratio.
- Use transparent PNG, WebP, or clean product photos with a simple background.
- If changing file extensions, update the matching `src` path in `index.html` and the relevant product detail page.
